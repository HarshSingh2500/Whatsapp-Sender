WhatsApp Message Sender

Instructions:

  • Messages are sent using WhatsApp Web, so ensure you're logged in on your default browser.
  • Each message is scheduled with a 2-minute buffer to allow WhatsApp Web to open and send it.
  • Messages are sent one by one, each with a delay of about 15–20 seconds.
  • Do not close the browser window that opens during the message-sending process.
  • The process runs in the background — you're free to return to the home page or upload another file.
  • Excel files should be in .xlsx format with correct column names:
    • Phone column for all files.
    • Message column for personalized messages.
  • Make sure phone numbers include country code, e.g., 919876543210
